## Sobre o projeto

Este repositório foi desenvolvido para a prova prática de Web Standards, unidade curricular do primeiro semestre de TSI ministrada pelo professor Correa.

O projeto tem como objetivo aplicar conceitos fundamentais de desenvolvimento front-end, utilizando HTML e CSS para construção e estilização de interfaces a partir de layouts propostos.

Durante o desenvolvimento, foram trabalhados conceitos como:

* Estruturação semântica
* Organização de layout
* Flexbox Layout
* Estilização com CSS
* Hierarquia visual
* Boas práticas de desenvolvimento front-end

## Objetivos acadêmicos

* Praticar a construção de interfaces modernas

* Desenvolver organização de código
* Trabalhar com layouts inspirados em protótipos do Figma

---

## Aprendizados

Este projeto contribuiu para o aprimoramento de habilidades relacionadas à estruturação de páginas web, estilização de componentes e organização visual de interfaces.

Além disso, serviu como prática para resolução de problemas comuns no desenvolvimento front-end, como alinhamento de elementos, utilização de Flexbox e aplicação visuais com CSS.
